# ITUI Design System — Overview

> **Unified UI for EchoIT Products**
> Design by ITUI 2025

---

## Purpose

ITUI is EchoIT's design system — built to deliver a consistent user experience and brand identity across all products and services. More than a component library, it's a shared foundation that aligns product, design, and engineering around a common language.

We're sharing ITUI because we believe good systems become more valuable — and more resilient — when they're used beyond a single team. We hope this serves as a useful reference for teams building their own design languages.

### Goals

| # | Goal | Description |
|---|------|-------------|
| ① | **Brand** | Every EchoIT product speaks with one voice. |
| ② | **Collaboration** | A shared standard that bridges roles and reduces friction. |
| ③ | **Efficiency** | Reusable components that accelerate development and simplify maintenance. |

---

## Structure

**Foundation → Atomic → Composite → Data Display → Feedback → Navigation & Overlay**

Each layer ensures consistency in design language, improves development efficiency, and supports scalable maintenance. Components within each category are organized in **alphabetical (A–Z) order**.

### Component Layers

| Layer | Description | Components |
|-------|-------------|------------|
| **Foundation** | Core design elements | Colors, Grid, Radius, Shadow, Spacing, Typography |
| **Atomic** | Smallest UI unit | Badge, Button, Checkbox, Chip, Divider, Input, Label, Radio, Slider, Toggle, Tag |
| **Composite** | Structure composed of multiple units | Accordion, Bubble, Calendar, Card, Carousel, Date Picker, Modal, Pagination, Pop-up, Progress, Scroll Bar, Stepper, Tab, Tooltip |
| **Data Display** | Information and data display elements | Empty State, List, Profile, Rating, Table |
| **Feedback** | Status and result notification elements | Backdrop, Skeleton, Snackbar, Spinner, Toast |
| **Navigation & Overlay** | Navigation, transition, and overlay interface | Bottom Sheet, Breadcrumb, Floating Button, GNB, Navigation, Overflow Menu, Popover, LNB |
