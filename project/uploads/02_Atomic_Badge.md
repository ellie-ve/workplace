# Atomic — Badge

> **Design by ITUI**
> 항목에 부가적인 상태나 수치를 덧붙여 보여주는 보조 표시 요소입니다.

---

## Variants

| Property | Values |
|----------|--------|
| **Type** | Badge, Circle, Overflow, Dot |

- **Badge** — 텍스트 레이블을 표시하는 기본형
- **Circle** — 숫자를 원형으로 표시 (예: `19`)
- **Overflow** — 최대값 초과 표시 (예: `100+`)
- **Dot** — 단순 점(dot) 표시, 텍스트 없음
