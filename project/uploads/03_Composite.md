# Composite — Accordion

> **Design by ITUI**
> 제목 영역을 기준으로 상세 내용을 펼치거나 숨길 수 있는 인터랙션 요소입니다.

---

## Variants

| Property | Values |
|----------|--------|
| **State** | Collapse, Expand |
| **Style** | Default, Filled, Ghost, Line, Outline |
| **Interaction** | Default, Hover, Pressed |

## Size Tokens

| Property | Token |
|----------|-------|
| Height | `{ height.accordion }` |
| Padding | `{ spacing.xl }` |
| Gap | `{ spacing.sm }` |
| Radius | `{ radius.sm }` |
| Icon | `{ height.icon.lg }` |
| Font Size | `{ font.size.md }` |
| Stroke | `{ stroke.xs }` |
| Width | `{ layout.width.hug }` |
| Auto Layout | `{ layout.auto.horizontal.center }` |
